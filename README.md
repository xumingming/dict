# dict

## Usage

```bash
xumingmingv:dict(git:master)$ lein repl
REPL started; server listening on localhost port 13475
dict.core=> (search "hello")
----- 发音 -----
       he'ləu, hə-
----- 标准解释 -----
       n. 表示问候， 惊奇或唤起注意时的用语
       int. 喂；哈罗
----- 网络解释 -----
       Hello  : 你好 ;您好 ;哈啰 ;
       Hello Kitty  : 凯蒂猫 ;昵称 ;凯帝猫 ;
       hello barbie  : 哈啰芭比 ;
nil
```
